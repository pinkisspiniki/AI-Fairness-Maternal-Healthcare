# AI Fairness in Maternal Healthcare
This repository contains the technical implementation and data analysis for my Ph.D. research at the **University of Hull**.

### Research Focus
The goal is to address the **'Operational Disconnect'** in healthcare AI by developing a full-stack tool that identifies **intersectional bias** in maternal health datasets. 

### Tech Stack
- **Backend:** Python (Fairlearn, AIF360, Pandas)
- **Frontend:** PHP (Visual Analytics Dashboard)
- **Methodology:** Counterfactual Fairness & Distributional Mitigation
